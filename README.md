# PartyGamesApp-Feedback

Welcome to the official feedback and issue tracking repository for **4x1 Party Games**! 🎮

This repository is a central hub for our community to suggest new features, report bugs, and help shape the future of the app.

---

## 🚀 How to Help

We value your feedback! To keep things organized, please use the following links to submit your input:

*   **[Report a Bug](https://github.com/beenabahuguna/PartyGamesApp-Feedback/issues/new?template=bug_report.md)**: Found something broken? Let us know so we can fix it.
*   **[Suggest a Feature](https://github.com/beenabahuguna/PartyGamesApp-Feedback/issues/new?template=feature_request.md)**: Have an idea for a new game or a UI improvement? We'd love to hear it!
*   **[General Discussion](https://github.com/beenabahuguna/PartyGamesApp-Feedback/discussions)**: Want to talk about game strategies or show some love? Start a discussion.

## 📱 About the App

**4x1 Party Games** is a premium, offline-first collection of your favorite party games, built for high-speed fun without the need for an internet connection.

*   **App Store**: [Download here](https://apps.apple.com/app/id6759181056)
*   **Privacy**: 100% on-device. No tracking. No ads.

---

## 🛠 Guidelines

*   **Search First**: Before opening a new issue, please search existing issues to see if it has already been reported.
*   **Be Descriptive**: Provide as much detail as possible (screenshots, device model, iOS version).
*   **Be Kind**: This is a community space for constructive feedback.
